🔢 How the NFL Formula Works

The official NFL passer rating formula uses four components, each capped between 0 and 2.375:

A = ( (Comp/Att) − 0.3 ) × 5
B = ( (Yards/Att) − 3 ) × 0.25
C = (TD/Att) × 20
D = 2.375 − ( (INT/Att) × 25 )
Rating = ( (A + B + C + D) / 6 ) × 100

The maximum possible rating is 158.3. A rating above 100 is considered excellent; 80–100 is above average; below 70 needs improvement.
